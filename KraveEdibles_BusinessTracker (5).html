Architecture Notes
How Google Sheets Backup Works:
1. Create a Google Apps Script project
2. Deploy as Web App (Anyone can access)
3. The script receives POST data and writes to separate sheets: Sales, Expenses, Ingredients, Recipes
4. Paste the /exec URL in settings above
5. Backup is triggered manually or auto on each sale

Security: Data sent via HTTPS POST. No OAuth needed for write-only Apps Script webhooks. Data is stored only in your own Google account.

📋 Apps Script Code to Copy:
function doPost(e) { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName(data.sheet) || ss.insertSheet(data.sheet); if (data.action === 'append') { sheet.appendRow(data.row); } else if (data.action === 'overwrite') { sheet.clearContents(); data.rows.forEach(r => sheet.appendRow(r)); } return ContentService .createTextOutput(JSON.stringify({ok:true})) .setMimeType(ContentService.MimeType.JSON); }
